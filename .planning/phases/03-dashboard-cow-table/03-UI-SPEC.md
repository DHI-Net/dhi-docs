# Phase 3 — UI Design Contract

> Visual and interaction contract for the farm dashboard and cow table. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (Tailwind CSS v4 utility-first, no component library) |
| Preset | not applicable |
| Component library | none (custom Vue 3 components) |
| Icon library | inline SVG arrows for KPI deltas; no external icon library required |
| Font | system font stack: `ui-sans-serif, system-ui, -apple-system, sans-serif` |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-text gap in KPI delta, alert badge internal padding |
| sm | 8px | Table cell padding, filter button gaps, badge margins |
| md | 16px | Card internal padding, section gaps, input field padding |
| lg | 24px | Card-to-card gap, filter bar padding, chart container padding |
| xl | 32px | Dashboard section breaks (KPI row to charts, charts to table) |
| 2xl | 48px | Top bar height internal padding |
| 3xl | 64px | Page top/bottom margins |

Exceptions: Top bar fixed height = 56px (multiple of 4, accommodates farm/month selectors comfortably).

---

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 14px | 400 (regular) | 1.5 | Table cells, filter labels, descriptive text |
| Label | 12px | 400 (regular) | 1.4 | KPI card titles, chart axis labels, column headers, badge text |
| Heading | 20px | 600 (semibold) | 1.3 | Section titles ("趨勢圖", "牛隻列表"), page heading |
| Display | 28px | 600 (semibold) | 1.2 | KPI card values (e.g., "33.4 kg") |

2 weights only: 400 (body/labels/table) and 600 (headings/KPI display values). Label text differentiates from Body by size (12px vs 14px), not weight.

1. Top bar background gradient (`#16a34a` to `#15803d`)
2. Active farm/month selector highlight
3. "Positive" KPI delta arrows (improvement direction)
4. Active filter button (selected parity/group)
5. Current page indicator in pagination

## Component Inventory

### TopBar (DASH-01)
- Fixed at page top, height 56px
- Green gradient background (`green-600` to `green-700`)
- Left: system name "DHI Net" in white, 20px semibold
- Center-right: Farm dropdown (white text, underline on hover) + Month dropdown
- Farm dropdown shows authorized farm names; month dropdown shows available months sorted descending
- Selected values stored in Pinia `farm` store
- On change: triggers full data reload for new farm-month

### KpiCard (DASH-02)
- White card, rounded-lg, shadow-sm, padding 16px
- Top: label in 12px medium gray-500
- Middle: value in 28px bold (Display role)
- Bottom: delta arrow (triangle up/down or dash) + absolute delta value in 14px
- Arrow color follows polarity map (see Color section)
- 4 cards in a row: grid-cols-4 on desktop, grid-cols-2 on tablet, grid-cols-1 on mobile

### KpiRow
- Wrapper component: CSS grid with 16px gap
- Contains exactly 4 KpiCard instances (乳量, SCC, 泌乳頭數, 空胎天數)

### FilterBar (FILT-01, FILT-02, FILT-03)
- Horizontal bar below KPI row, background gray-50, padding 16px, rounded-lg
- Three filter groups arranged horizontally with 24px gap between groups
- Each group: label (12px medium) above a row of pill buttons
- Parity filter: 3 buttons (全部 / 頭產 / 經產)
- Group filter: dynamic buttons extracted from cow data (全部分群 + each unique group)
- Time range: 4 buttons (近 3 個月 / 近 6 個月 / 近 12 個月 / 自訂範圍)
- Active button: green-600 background, white text
- Inactive button: white background, gray-700 text, gray-200 border
- All filter changes write to Pinia `filter` store; no API calls

### TrendChart (DASH-03)
- ECharts line chart, height 300px, autoresize
- X-axis: month labels (e.g., "2025.7", "2025.8", ... "2026.6")
- Dual Y-axis: left for 乳量 (kg), right for SCC (萬/mL)
- 3 series: 乳量 (blue line), SCC (red line), 泌乳頭數 (green line)
- Tooltip: axis trigger showing all 3 values
- Legend: top-center, horizontal
- Responds to FILT-03 time range filter

### ParityChart (DASH-04)
- ECharts bar chart, height 250px, autoresize
- Vertical bars: 頭產, 二胎, 三胎以上
- Single Y-axis: head count
- Colors: blue-400, emerald-400, amber-400
- Computed client-side from `cows[].parity`
- Responds to FILT-02 group filter (but NOT parity filter, since it IS the parity view)

### SccDistChart (DASH-05)
- ECharts bar chart, height 250px, autoresize
- Vertical bars for 4 SCC ranges: <5, 5-20, 20-50, >50 (萬/mL)
- Colors: green-600, blue-400, amber-500, red-600 (severity gradient)
- Computed client-side from `cows[].scc`
- Responds to FILT-01 parity filter and FILT-02 group filter

### CowTable (COW-01, COW-02, COW-03, COW-04)
- Full-width table with horizontal scroll on mobile
- Header row: gray-50 background, 12px medium text, sticky top
- Clickable column headers with sort indicator (triangle up/down)
- Default sort: 場內編號 ascending
- Search input above table: full-width, border gray-300, rounded, 14px body text
- Search filters on `farm_id` and `national_id` fields (client-side, instant)
- Alert badges: inline pill badges per row, colored per Alert Badge Colors table
- Multiple alerts on one cow: badges displayed inline with 4px gap
- Alternating row backgrounds: white / gray-50
- Hover state: gray-100 background
- Pagination below table: showing count text + prev/next buttons
- 50 rows per page; client-side pagination

## Visual Hierarchy

**Primary focal point:** The KPI row (平均乳量, SCC, 泌乳頭數, 空胎天數) is the primary visual anchor — the 28px semibold values draw the eye before charts or table. Eye flow: TopBar identity → KPI metrics → FilterBar → supporting charts → detail table.

## Interaction States

### Loading
- Skeleton placeholders for KPI cards (gray-200 pulse animation)
- Chart containers show centered "資料載入中..." text in gray-400
- Table shows 5 skeleton rows

### Empty
- All KPI cards show "--" for value, no delta arrow
- Charts show empty state with centered text: "尚無資料"
- Table shows single row spanning all columns: "此牧場尚無測乳紀錄。請先上傳 PDF 報表以匯入資料。"

### Error
- Red-bordered alert box at top of dashboard content area
- Text: "資料載入失敗，請檢查網路連線後重新整理頁面。"
- Retry button: "重新載入資料" with green-600 background

### Filter Active
- Active filter pill: green-600 background, white text, rounded-full
- Inactive filter pill: white background, gray-700 text, gray-200 border, rounded-full
- Hover on inactive: gray-50 background
- Transition: 150ms ease background-color

### Table Sort
- Unsorted column: no indicator
- Ascending: small triangle up icon after column text (gray-400 -> gray-700 when active)
- Descending: small triangle down icon after column text
- Click toggles: unsorted -> ascending -> descending -> ascending

### Pagination
- Current page button: green-600 background, white text
- Other page buttons: white background, gray-700 text, gray-200 border
- Disabled prev/next: gray-300 text, cursor-not-allowed

## Accessibility

- All chart containers include `aria-label` with chart description (e.g., "12個月乳量、體細胞數與泌乳頭數趨勢圖")
- Table uses proper `<table>`, `<thead>`, `<tbody>`, `<th scope="col">` semantic HTML
- Sort buttons on `<th>` elements have `aria-sort` attribute (ascending/descending/none)
- Filter buttons use `role="radiogroup"` for each filter group, `aria-pressed` on each button
- Color is never the sole indicator: delta arrows use shape (triangle up/down/dash) in addition to color
- Search input has associated `<label>` (visually hidden if needed)
- Minimum touch target size: 44x44px for all interactive elements on mobile
